This is the Utility folder of the Project.
All config files will be stored here.

'secrect.py' = Enter your Instagram Username and Password
'hashtags.py' = Enter the hashtags you want to use for the Bot
'limits.py' = Enter the scripts rate limits for likes, comments, stories and follows
'proxy.py' = Enter your proxy settings here
'config.py' = Choose what modules to run (like, comment etc)
'userlist.py' = User list for story scraping (mass story viewer)
